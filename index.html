Python in BrainVoyager

Starting with BrainVoyager 20.0, experimental Python support is available as an easy and unified alternative to both standard JavaScript scripting and C++ plugin development.

The embedded Python interpreter is enriched with several extra packages (e.g. Numpy, Scipy, PythonQt) and provides three interfaces to internal components of BrainVoyager:

BrainVoyager script access

This is basically a wrapper for the “old” JavaScript scripting API.

The interface can be accessed through the readily available object BrainVoyager, which is an instance of PythonQt.private.BrainVoyagerScriptAccess.

BrainVoyager array data

This is a speed optimized interface to internal BrainVoyager data working
with Numpy arrays.

After calling import bv, the interface can be accessed through the module bv.

Graphical user interfaces

This is an interface to the Qt GUI framework and can be used to create cross-platform graphical user interfaces.

After calling import PythonQt.Qt the interface can be accessed through the module PythonQt.Qt.
